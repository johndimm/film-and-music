Atomic vs Composite (closest prior framings) + bipartite “two-mode” graphs

Your Atomic↔Composite split is a practical UI/interaction constraint. Prior work typically uses different names: actor↔event (cultural heritage / museum ontologies), continuant↔occurrent (upper ontologies), and two-mode / affiliation networks (social network analysis).

  • CIDOC CRM (ISO 21127) — event-centric cultural heritage ontology (Actor/Event modeling).
    museum informatics event-centric modeling actor ↔ event
    Spec / site: cidoc-crm.org
  • Stead, S. (2008). CIDOC CRM Tutorial (slides) — includes “Historical events as meetings”.
    event-centric modeling people ↔ events museum informatics
    Slide deck: CIDOC CRM Tutorial (SlideShare)
  • Basic Formal Ontology (BFO) — “continuants” vs “occurrents” (things vs happenings/events).
    upper ontology continuant ↔ occurrent
    Site: basic-formal-ontology.org
  • Two-mode / affiliation networks — classic bipartite modeling in social network analysis.
    two-mode networks bipartite graphs people ↔ events
    Practical visualization tool (supports 2-mode networks): Pajek (widely used in SNA; includes 2-mode network operations and visualization).
  • Borgatti, S. P. (c. 1990s–2000s). Network Analysis of Two-Mode Data.
    two-mode networks affiliation networks methods
    PDF: Network Analysis of Two-Mode Data (ResearchGate) · Useful for motivating bipartite-specific ranking/weighting (e.g., avoid projection artifacts) without making causal claims.
  • Faust, K. (2005). Using Correspondence Analysis for Joint Displays of Affiliation Networks.
    joint display 2D embedding affiliation networks
    PDF: faust_correspondence_analysis_2005.pdf · Directly relevant to “map-like wandering”: places actors and events in a shared space via correspondence analysis and discusses what “near” means.
  • Snijders, T. A. B. (2025). Two-mode Network Dynamics (slides/notes).
    definitions two-mode networks dynamics (background)
    PDF: Twomode_s.pdf · Mostly relevant for standard terminology and bipartite structures (e.g., 4-cycles). The SAOM/dynamics machinery assumes access to a global observed two-mode matrix, which differs from Constellations’ LLM-generated, local exploration setting.
  • Two-mode / affiliation networks (reference PDF: socnet07.pdf).
    two-mode networks affiliation networks bipartite
    PDF: socnet07.pdf · Note: metadata/title extraction is unclear; treat as a “lead” and we can replace with a canonical citation once identified.
