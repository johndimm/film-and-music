Fix Wikipedia exchars parameter silently under-delivered text for citation-heavy articles — Gayle Skidmore's article returned only 1,208 chars regardless of the limit requested. Fix Stale is_atomic values in the cache caused the bipartite filter to drop all edges for nodes like The Godfather — actors were stored with is_atomic: false. Now infers correct atomicity from the parent node's bipartite position when loading from cache.
