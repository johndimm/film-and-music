🌌 Support & Help

Everything you need to know about Constellations

Getting Started

What is Constellations?

Constellations is an AI-powered knowledge explorer that creates interactive graphs showing how entities are connected. It works across any domain - from world history to cooking, academic research to sports.

How do I start exploring?

Simply search for any topic, person, or concept you're interested in. The AI will:

  1. Identify the appropriate knowledge domain (e.g., Person ↔ Event, Ingredient ↔ Recipe)
  2. Find related entities and build a graph around your starting point
  3. Let you expand nodes to discover more connections

Features & How to Use

What are "Atomic" and "Composite" nodes?

Atomic nodes (shown as circles) are fundamental building blocks like people, ingredients, or symptoms. Composite nodes (shown as cards) are collections that bring atomics together - like events, recipes, or diseases.

This bipartite structure prevents false connections and keeps the graph logically organized.

How do I expand a node?

Click on any node to select it. If it hasn't been expanded yet, it will automatically find and display connected entities. For nodes already expanded, you can click "Expand More" to reveal additional connections.

What is Timeline mode?

Timeline mode arranges nodes chronologically instead of as a network. This is particularly useful for historical exploration, showing events and people in temporal order.

How does Path Finding work?

Switch to "Connect" mode (instead of "Explore") to find paths between any two entities. Enter a start and end point, and the AI will discover the shortest connection path, expanding the graph as needed.

Supported Domains

Constellations automatically detects and works across many domains, including:

  • History: Person ↔ Event
  • Culinary: Ingredient ↔ Recipe
  • Medicine: Symptom ↔ Disease
  • Academia: Author ↔ Paper
  • Film: Actor ↔ Movie
  • Music: Musician ↔ Album
  • ...and more!

Troubleshooting

I got "No connections found" - what should I do?

Try being more specific with your search term. For example, instead of just "Smith", try "Will Smith" or "Adam Smith (economist)". You can also try a different starting point.

An image looks wrong - can I fix it?

Yes! Click the node, then use the "Find Better Image" option in the context menu. The AI will search for a more appropriate photo.

Can I save my graphs?

Yes! Use the "Save Graph" button to store your current exploration. You can load it later or export it as an image or JSON file.

Privacy & Data

What data is collected?

Constellations caches AI responses and graph data to improve performance and reduce API costs. No personal information or browsing history is collected. All data is stored securely and used only to enhance your exploration experience.
